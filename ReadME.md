# Travel-Agency-App
Minimal sample app for a travel agency.


## Technologies Used

* JPA/Hibernate for the persistence layer
* MySQL, H2
* Lombok
* Javax validation
* Spring Security, JWT
* Slf4j Logging
* Swagger3
* JDK 11
* Maven
* JUnit5


### Running the application locally

```shell
mvn clean install

mvn spring-boot:run
```


### Running the tests

```shell
mvn test
```
